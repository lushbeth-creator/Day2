How to Start an Application

Using the Employee Portal, start by finding the Operations tab at the top. This is where you will find all the resources you need to start filling out a Merchant application, adding or replacing equipment on an existing account, and many other helpful links.

Important things to note during this step:
  • Make sure to complete the equipment section thoroughly, paying extra attention to the parameter notes, punctuation, capitalization, and finance options
  • Any Clover menu builds, ACH forms, etc. need to also be attached to the application
